About DAH Files

                
DAH (Designated Airspace Handbook) files contain airspace information from Air Services Australia including:

                
    
                    
  • Airspace boundaries and volumes
    • 
                    
  • Upper and lower altitude limits
    • 
                    
  • Airspace identification and names
    • 
                    
  • Coordinate information (latitude/longitude)
    • 
                

                
This tool extracts this data and converts it to the VATGlasses JSON format for use with VATGlasses applications.
